Description of AuderoWavExtractor

PHP versions 4 and 5

The follow is an example of how to extract a chunk from a wav file and force the download using this class:

  1.  <?php
  2.     require_once('AuderoWavExtractor.php');
  3.  
  4.     $InputFile = 'audio.wav';
  5.     $OutputFile = 'chunk.wav';
  6.     $Start = 0 * 1000;
  7.     $End = 20 * 1000;
  8.  
  9.     $Extractor = new AuderoWavExtractor($InputFile);
  10.     $Extractor->extractChunk($Start, $End);
  11.  ?>

The follow is an example of how to extract a chunk from a wav file and save it into the local disk:

  1.  <?php
  2.     require_once('AuderoWavExtractor.php');
  3.  
  4.     $InputFile = 'audio.wav';
  5.     $OutputFile = 'chunk.wav';
  6.     $Start = 0 * 1000;
  7.     $End = 20 * 1000;
  8.  
  9.     $Extractor = new AuderoWavExtractor($InputFile);
  10.     $Extractor->extractChunk($Start, $End, 2, $OutputFile);
  11.  ?>
