		extractChunk (line 224)
	
 
	
	

Extract a portion of the current, from the $Start to $End. The chunk will be  saved, with name $Filename, in the local server, the browser or the user and  so on based on the value of $Destination.


The possible value of $Destination are:  
  • 1: send the file to the browser and force the download with the
       name given by $Filename.
  • 2: save into the local disk the file with the name given by $Filename.
  • 3: return the wav as a string. In this case $Filename is ignored.
  • 4: both 2 and 3

	
    
				
  • return: The chunk to extract, including the headers
    • 
				
  • access: public
    •
