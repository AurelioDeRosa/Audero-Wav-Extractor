    This class represents the value of a chunk. Although a value in the chunks is always a string, which of course can also represent an integer, I created a whole class to keep track of other info about the value. Infact, every value has potentially a different format (used by pack and unpack function of PHP to extract the value - more info here: http://php.net/manual/en/function.pack.php) and size (in bytes).
