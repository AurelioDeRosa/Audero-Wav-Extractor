    "Audero Wav Extractor" is a PHP library that allows to extract a chunk from a wav file. The extracted chunk of the wav file can be saved on the hard disk, can be forced to be prompted as download by the user's browser, returned as a string for a later processing or a combination of the first and second possibilities. It is very easy to use "Audero Wav Extractor" to extract a piece of audio from a wav file. All you have to do is give the name of the file, the start and the end time to extract (optionally you can provide a name for the extracted chunk).

    The follow is an example of how to extract a chunk from a wav file and force the download using this class:

       require_once('AuderoWavExtractor/AuderoWavExtractor.php');

   $InputFile = 'audio.wav'; // path to input wav file
   $OutputFile = 'chunk.wav'; // path to the output chunk
   $Start = 0 * 1000; // Start time of the chunk
   $End = 20 * 1000; // End time of the chunk

   $Extractor = new AuderoWavExtractor($InputFile);
   $Extractor->extractChunk($Start, $End);

    The follow is an example of how to extract a chunk from a wav file and save it into the local disk:

       require_once('AuderoWavExtractor.php');

   $InputFile = 'audio.wav'; // path to input wav file
   $OutputFile = 'chunk.wav'; // path to the output chunk
   $Start = 12 * 1000; // Start time of the chunk
   $End = 20 * 1000; // End time of the chunk

   $Extractor = new AuderoWavExtractor($InputFile);
   $Extractor->extractChunk($Start, $End, 2, $OutputFile);

    Extract a portion of the current, from the $Start to $End. The chunk will be saved, with name $Filename, in the local server, the browser or the user and so on based on the value of $Destination.

    extractChunk(int $Start, int $End, int $Destination, string $Filename) : string

    The possible value of $Destination are:

    • 1: send the file to the browser and force the download with the name given by $Filename.
    • 2: save into the local disk the file with the name given by $Filename.
    • 3: return the wav as a string. In this case $Filename is ignored.
    • 4: both 2 and 3

    Parameters

    $Destination

    Where to save the extracted chunk

    $Filename

    string

    &$Filename The name to assing to the chunk. If no name is provided the method generate one using this rule: "InputFilename-Start-End.wav"

    Exceptions

    \InvalidArgumentException If the give boundary is invalid (for example if the end is lower than start) or if the destination value isn't valid (for example greater than 4).
    \RuntimeException If there's not enough memory to save the given range
